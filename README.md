# Ey App - A Name Badge for Northerners

Git tha Tildagon wekkin' as a badge fut gatherin, s' folk can tell thi from Adam.

*(Use your Tildagon as a conference name badge to introduce yourself in a northern dialect.)*

## Features

* Many northern greetings.
* Adjustable chaos level.
* Makes you smart and attractive.

## Controls

* A/D buttons: Increase/Decrease chaos level.
* B button: Show framerate.
* F button: Suspend Ey App and return to the Tildagon main menu.

## Install

### From App Store

Available from the app store: https://apps.badge.emfcamp.org/apps/11001423/

### Local Install

Cross compile with the `mpy-cross` tool and download to the badge with the `mpremote` tool:

```
mpy-cross app.py
mpremote fs cp app.mpy :/apps/mbooth101_emf_eyapp
```
